Privacy Policy

Last updated: March 24, 2026

This Privacy Policy applies to Markdown Manager for Google Docs™, a Google Workspace Add-on.

1. Data we access

This add-on requests the following OAuth scopes:

  • https://www.googleapis.com/auth/documents — read and write the content of the active Google Doc. This is required to parse the document structure, navigate to elements, and reorder sections when the user drags a node in the sidebar.
  • https://www.googleapis.com/auth/script.container.ui — show the sidebar and modal dialogs inside Google Docs.

2. How data is used

Document content is used exclusively to render the in-app UI (sidebar tree, Markdown preview). Data is processed entirely within Google's Apps Script infrastructure and is never transmitted to any external server, database, or third party.

We do not:

  • sell your data to any third party;
  • use your data for advertising or any purpose unrelated to the add-on's stated features;
  • allow humans to read your document content, except as required by law.

Data access is strictly limited to what is necessary for the add-on to function.

3. Data storage

The application does not store or persist any document content or personally identifiable information. No analytics, cookies, or tracking technologies are used.

The only value stored server-side is your theme preference (light / dark / system), saved via Google's PropertiesService and associated with your Google account. It contains no document data and can be cleared by uninstalling the add-on.

To revoke this add-on's access to your Google account at any time, visit myaccount.google.com/permissions and remove Markdown Manager for Google Docs™.

4. Third-party services

The rendered Markdown preview loads the following resources from public CDNs at render time:

  • markdown-it and plugins — cdn.jsdelivr.net
  • highlight.js — cdnjs.cloudflare.com
  • Google Fonts (Roboto Mono, Material Icons Round) — fonts.googleapis.com

These requests are made by the user's browser directly to those CDNs. All external resources are loaded with Subresource Integrity (SRI) verification — if a CDN asset is tampered with, the browser will refuse to load it. Please review their respective privacy policies for information on how they handle request data.

5. Google API Services User Data Policy

Markdown Manager for Google Docs™'s use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

6. Contact

For privacy-related questions, contact the developer at info@riccardobianconi.it or open an issue on the GitHub issue tracker.
