Markdown Manager for Google Docs™

Getting started

Install the add-on from the Google Workspace Marketplace, then open any Google Doc. Go to Extensions → Markdown Manager for Google Docs™ → Open Markdown Manager to launch the panel.

Sidebar overview

The sidebar displays a live structural tree of your document based on Markdown syntax. Headings, lists, code blocks, and tables are all detected automatically.

Navigating the document

Click any item in the sidebar tree to jump directly to that element in the document.

Reordering sections

Drag a heading node in the sidebar to move the corresponding section (and all its content) to a new position. This operation modifies the document and can be undone with Ctrl+Z / ⌘Z.

Markdown preview

Click the Markdown Preview button at the top of the sidebar to open a full-document preview rendered as formatted Markdown — with syntax-highlighted code blocks, tables, footnotes, and more.

Theme

Use the theme button in the sidebar footer to switch between three modes: System (follows your OS preference automatically), Light, and Dark. Your choice is saved per Google account and persists across sessions.

For questions or bug reports, open an issue on the GitHub issue tracker.
